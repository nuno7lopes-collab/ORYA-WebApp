# Discover: Web vs App (Diferenças Intencionais)

## Princípio
Paridade por objetivo: ambas resolvem os jobs principais (descobrir, filtrar, decidir, agir), sem exigir paridade visual/estrutural 1:1.

## Diferenças aprovadas
1. Mapa interativo
   - App: capability principal (mapa completo).
   - Web: CTA explícito para app (`orya://map`) + alternativa web.
2. Navegação interna Discover (app)
   - App mantém modelo atual por decisão explícita.
   - Web mantém mundos canónicos `Eventos / Torneios / Reservas`.

## Igualdades obrigatórias mantidas
1. Filtros essenciais (`city`, `date`, `price`) com semântica equivalente.
2. Personalização por sinais autenticados (click/view/dwell/favorite/hide).
3. Estado vazio orientado à recuperação (vazio assistido).
